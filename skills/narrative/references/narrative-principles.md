# Narrative Strategy Principles

Shared knowledge base for the Narrative Strategist agent. Distilled from 52-agent research swarm covering Duarte, Ogilvy, Sutherland, Miller, Schwartz, Cialdini, Campbell/Vogler, Wiebe, Handley, Clark, and 7 proprietary research PDFs.

---

## 1. Schwartz's 5 Awareness Levels

The diagnostic engine. Determines everything else.

| Level | Audience State | Headline Strategy | Page Structure |
|---|---|---|---|
| Unaware | No problem recognition | Identity/aspiration hook | Short, evocative, no product |
| Problem-Aware | Feel pain, seeking | Empathy + agitation | Problem validation → intrigue |
| Solution-Aware | Comparing options | Differentiation + mechanism | Feature comparison + proof |
| Product-Aware | Know you, hesitant | Objection handling | Testimonials + guarantees |
| Most-Aware | Ready to buy | Clear offer + urgency | CTA + price + final push |

## 2. The Duarte Digital Sparkline

Scroll depth (Y-axis) mapped to emotional state (X-axis: Status Quo vs Visionary Future).

- The sparkline MUST oscillate — 3+ consecutive sections at the same pole = flatline = dropout
- Hero = Visionary Future flash (hook + promise)
- Problem sections = Status Quo pole (validate pain)
- Solution sections = Visionary Future pole (mechanism of change)
- STAR Moment = peak intensity at 60-70% depth
- CTA = curated resolution (never a generic footer)

Oscillation frequency: fast (~600px/cycle) for simple B2C, slow (~1800px/cycle) for complex B2B.

## 3. Sutherland's Psycho-Logic

- **Perceived value > Objective value.** Reframing changes everything without changing the product.
- **Satisficing > Maximizing.** Users seek "least likely to be terrible," not "objectively best." Emphasize risk reduction.
- **Context shapes meaning.** Small copy/framing changes create outsized effects. One client tripled sales with 3 sentences added to a script.

## 4. The StoryBrand Framework (Miller)

7-part structure: Character (customer) → Problem (external/internal/philosophical) → Guide (your brand) → Plan (clear path) → Call-to-Action → Success (outcome) → Avoid Failure (stakes).

The BrandScript one-liner: Who you help + What you do + Specific benefit.

**Critical inversion:** Brand = Mentor (Yoda), Customer = Hero (Luke). Never position the product as the protagonist.

## 5. Cialdini's 7 Principles for Web

| Principle | Web Pattern |
|---|---|
| Reciprocity | Free trials, guides, tools before asking |
| Commitment | Email opt-ins, micro-commitments, checkboxes |
| Social Proof | Testimonials with photos/names, user counts, ratings |
| Authority | Trust badges, expert endorsements, certifications |
| Liking | Relatable customer stories, conversational tone |
| Scarcity | "Only 3 left," countdown timers, limited access |
| Unity | "People like you," community language, shared identity |

**Pre-Suasion:** The moment BEFORE persuasion shapes receptivity. Visual priming (imagery, colors, background) prepares the mind before the argument lands.

## 6. The Hero's Journey (Minimum Viable)

5 beats for a landing page:
1. Ordinary World (their current pain)
2. Call to Adventure (your solution exists)
3. Mentor Introduced (your brand as guide)
4. Transformation (proof of results)
5. Threshold Crossed (CTA — commit to the journey)

## 7. Peak-End Rule (Kahneman)

Users judge experiences by the Peak (most intense moment) and the End (how it concluded). Design for both:
- **Engineer positive peaks** — Success States, STAR Moments, celebratory onboarding
- **Mitigate negative peaks** — Gamified 404s, engaging loading states
- **Curate the ending** — The final section/CTA is the second thing users remember

## 8. Cognitive Biases (Ethically Applied)

- **Anchoring:** Show higher-priced option first to make the target option feel valuable
- **Loss aversion:** Frame around what they lose by not acting (32% conversion lift)
- **Framing effect:** Loss-framed copy outperforms gain-framed (93% vs 67% in one study)
- **Default effect:** Pre-select beneficial options
- **Dark pattern boundary:** Never use fake scarcity, manufactured proof, or obstructed exits

## 9. The Glanceability Framework (Non-Narrative Products)

For dashboards, admin panels, developer tools:

**Consumption Speed Hierarchy:**
| Mode | Time | User Intent | UI Analog |
|---|---|---|---|
| Glance | 0-3s | "Is the system healthy?" | Status badges, hero KPIs |
| Scan | 3-10s | "What needs attention?" | Data tables, card grids |
| Study | 10s+ | "What's the root cause?" | Reports, drill-downs |

**Glanceability Audit Checklist:**
- Focal Singularity: one dominant element per screen (eye lands in 0.5s)
- 5-Word Test: primary utility describable in 5 words
- Contrast Hierarchy: secondary elements 30%+ lighter/smaller than primary
- Semantic Integrity: color used for meaning, not decoration
- Whitespace Ratio: 24-32px between functional zones

**Dashboard Narrative = State Machine:**
Act I (Context): establish baseline — sparklines, deltas, calm neutral state
Act II (Conflict): threshold breach — red/amber status, items reorganize, gap visualization
Act III (Resolution): clicking conflict surfaces action tools — the narrative loop closes

**Tufte vs Holmes:** Monitoring screens = Tufte (strip everything). Persuasion screens = Holmes (embellishment aids memory).

For scrollytelling and brand experiences — each mode changes palette, pacing, tone:

|---|---|---|---|---|
| Fincher | Crushed black, sodium vapor, teal | Procedural, dense, precise | Clinical dread, entropy | Problem/tension scenes |

## 11. Voice-of-Customer Research Protocol

Before writing, mine customer language from:
1. **Reviews** (Amazon, G2, app stores) — extract pain points, adjectives, desired outcomes
2. **Support tickets** — identify friction vocabulary
3. **Reddit/forums** — unprompted language about the problem space
4. **Customer interviews** — "What word would you describe this experience?"

Use THEIR words in headlines, value propositions, and CTAs. Copy that mirrors customer language creates authenticity.

## 12. Cross-Cultural Considerations

- Low-context cultures (US, Germany, Scandinavia): explicit, direct, data-driven
- High-context cultures (Japan, China, Middle East): implied meaning, relationship-first, visual richness
- Color meanings vary: white = purity (West) / mourning (East), red = danger (West) / luck (East)
- Blue is universally positive across cultures
- When in doubt, test with the target audience — never assume
