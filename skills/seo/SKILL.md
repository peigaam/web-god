# SEO Skill

Audit and optimize for search engine visibility.

## Pipeline
1. **Crawlability** — robots.txt, sitemap, internal linking, redirects
2. **Meta Tags** — Title, description, canonical per page
3. **Structured Data** — JSON-LD schemas per page type
4. **Open Graph** — Social sharing meta tags
5. **Content Structure** — Heading hierarchy, alt text, anchor text
6. **Performance** — Core Web Vitals (Google ranking factors)
7. **Report** — Per-category score /10 with prioritized fixes
