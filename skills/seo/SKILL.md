# SEO Skill

Audit and optimize for search engine visibility.

## Pipeline
1. **Crawlability** — robots.txt, sitemap, internal linking, redirects
2. **Meta Tags** — Title, description, canonical per page
3. **Structured Data** — JSON-LD schemas per page type
4. **Open Graph** — Social sharing meta tags
5. **Content Structure** — Heading hierarchy, alt text, anchor text
6. **Performance** — Core Web Vitals (Google ranking factors)
7. **Report** — Per-category score /10 with prioritized fixes

## Review Checklist
- [ ] Every page has unique title (50-60 chars) and description (150-160 chars)
- [ ] Canonical URLs set on all pages
- [ ] JSON-LD structured data matches page content
- [ ] XML sitemap lists all indexable pages
- [ ] Core Web Vitals within good thresholds
- [ ] Open Graph tags include image (1200x630 min)

**Human checkpoint:** Present the SEO audit report with per-category scores for approval before implementing fixes.

Consult `references/seo-principles.md` for domain-specific guidelines.
